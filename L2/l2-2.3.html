Назад Функции памяти

Функции памяти

  1. Прием информации от других устройств
  2. Хранение информации
  3. Передача информации по запросу в другие устройства комьютера

Память делят на:

Основную(или внутреннюю)

    ОЗУ (оперативное запоминающее устройство), по-английски RAM (Random Acces Memory - "память с произвольным доступом")
    ПЗУ (постоянное запоминающее устройство), по-английски ROM) (Read Only Memory - "память только для чтения")
Внешнюю (устройства внешней памяти позволяют длительно хранить информацию)
    Носители внешней памяти: жесткие и гибкие магнитыне диски, а также лазерные диски и флеш-карты
